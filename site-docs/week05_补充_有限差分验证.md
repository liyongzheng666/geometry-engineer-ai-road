# 第 5 周补充：有限差分验证——中心差分与步长选择

> 配套代码：[`code/week05_A/energy.h`](https://github.com/liyongzheng666/geometry-engineer-ai-road/tree/main/code/week05_A)
> 这篇把本周的"裁判"讲透：为什么用中心差分、步长 h 怎么选、为什么 Hessian 要
> 差分梯度而不是二阶差分能量、以及相对误差判据的两个坑。

## 一、前向差分 vs 中心差分：一个量级的差距

近似导数最朴素的是**前向差分**：

```text
f'(x) ≈ (f(x+h) − f(x)) / h            截断误差 O(h)
```

**中心差分**多算一个点，误差降一个阶：

```text
f'(x) ≈ (f(x+h) − f(x−h)) / (2h)       截断误差 O(h²)
```

代价只是多一次函数求值，精度却从 O(h) 到 O(h²)。验证解析梯度这种"要卡到 1e-6"的场景，
中心差分是标配。`energy.h` 的 `fd_gradient` / `fd_hessian_from_grad` 都用它。

## 二、步长 h 的两难：截断误差 vs 舍入误差

h 不是越小越好。总误差是两股力量的拉锯：

```text
截断误差 ~ C₁·h²        （h 越大越糟）
舍入误差 ~ C₂·ε/h       （h 越小越糟，ε≈2.2e-16 是机器精度）
```

两者相加在某个 h* 处最小。对中心差分，最优步长约 `h* ~ ε^(1/3) ≈ 6e-6`，
此时总误差 ~ 1e-11 量级。这正是 `fd_gradient` 默认 `h=1e-6` 跑出相对误差 ~1e-10 的原因。

| h 取值 | 主导误差 | 后果 |
|---|---|---|
| 1e-1 | 截断 O(h²)=1e-2 | 太粗，假 FAIL |
| **1e-6** | 平衡点附近 | **~1e-10，刚好** |
| 1e-12 | 舍入 ε/h=1e-4 | 灾难性抵消，反而更差 |

**经验法则**：一阶导数 `h ≈ ε^(1/3) ≈ 1e-5~1e-6`，别手贱调到 1e-10。

## 三、为什么 Hessian 差分梯度，而不是二阶差分能量

二阶导直接差分能量是这样：

```text
f''(x) ≈ (f(x+h) − 2f(x) + f(x−h)) / h²      舍入误差 ~ ε/h²
```

注意分母是 **h²**。h=1e-5 时舍入误差 ~ 2.2e-16/1e-10 = **2e-6**——这点精度根本卡不住
一个要求 1e-5 的 Hessian。而且分子是三个相近数相减，灾难性抵消更严重。

`fd_hessian_from_grad` 改成**对解析梯度做一次中心差分**：

```text
H[i][j] ≈ (g_i(p+h e_j) − g_i(p−h e_j)) / (2h)      舍入误差 ~ ε/h ≈ 1e-11
```

分母回到 h（不是 h²），精度稳两个量级。代价是它依赖"梯度已经正确"——所以验证有顺序：
**先用能量差分钉死梯度（test_gradient），再用梯度差分钉 Hessian（test_hessian）。**
这个"验证链"是本周最容易被忽略的工程细节。

## 四、相对误差判据的两个坑

`test_gradient` 的判据是：

```cpp
rel = |a − b| / max(1.0, |a|, |b|)
```

- **坑一：零点附近**。若真值接近 0，纯相对误差 `|a−b|/|b|` 会因分母趋零而假性爆炸。
  分母兜底 `max(1.0, …)` 把它转成"接近 0 时看绝对误差"，避免误报。
- **坑二：量级混合**。梯度两个分量量级可能差很多，逐分量取 `max` 比"整体范数比"更严——
  一个分量错了不会被另一个大分量稀释掉。

## 五、把有限差分当"回归测试"

有限差分验证不是一次性的。它值钱在**可回归**：

- 你改了能量模型 → 重跑，立刻知道梯度/Hessian 还对不对；
- AI 换了个"更简洁"的 SymPy 写法 → 重跑，确认没引入符号错；
- 第 6 周做 CSE 优化 → 重跑，确认优化没改变数值结果。

这就是为什么本周坚持把验证放进 C++ harness 而不是只在 Python 里跑一次：
**它会在后面每一周继续当裁判。** 手推公式给不了这种可回归的安全网。

> 一句话：**中心差分是几何数值代码最便宜的保险**——多写十行 FD，
> 换来"AI 推导 / 你手改 / 后续优化"三种改动下都有人替你验对错。
